# Timber-Project-2-

This project is a simple yet engaging implementation of a Timber-style arcade game developed in C++. Inspired by classic reflex-based games, the player takes on the role of a woodcutter who must chop a tree as quickly as possible while avoiding incoming branches.

The project focuses on building core game development concepts such as real-time input handling, collision detection, game loops, animation timing, and score tracking. It is structured to keep the code modular and readable, making it ideal for beginners who want to understand how interactive games are built from scratch.

With progressively increasing difficulty and responsive controls, the game offers both a fun gameplay experience and a practical demonstration of fundamental programming and design principles.
